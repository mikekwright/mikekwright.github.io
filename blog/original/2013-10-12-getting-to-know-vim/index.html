Useful Shortcuts

  • Disable Search Highlighting

      set nohlsearch   " Turn it off
  set hlsearch!    " Toggle on or off

  :nohlsearch      " Turn it off as command
  :nohl            " Shorthand
  :noh             " Even Shorter
